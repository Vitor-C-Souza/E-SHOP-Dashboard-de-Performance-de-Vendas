# E-SHOP: Dashboard de Performance de Vendas
## 🎯 Objetivo do Projeto
Este projeto foi desenvolvido como parte de um desafio de análise de dados, com o foco em transformar dados brutos de vendas em informações visuais claras e úteis para a tomada de decisão. O dashboard permite uma análise eficaz do desempenho de vendas, lucros e eficiência operacional do e-commerce E-SHOP.

## 📊 Principais Insights e Métricas
O dashboard consolida os seguintes indicadores de performance:

- **Faturamento Total:** R$ 482.308,84.

- **Lucro Líquido Total:** R$ 51.682,92.

- **Margem de Lucro Média:** 10,72%.

- **Ticket Médio por Pedido:** R$ 375,34 (baseado em 1.285 pedidos únicos).

- **Volume de Vendas:** 7.033 produtos comercializados.

- **Liderança Regional:** A região West destaca-se como a mais lucrativa, gerando R$ 18.992,67 em lucro.

## 🛠️ Tecnologias e Recursos Utilizados
- **Microsoft Excel:** Utilizado para processamento de dados e interface visual.

- **Tabelas Dinâmicas:** Para a agregação e tratamento de grandes volumes de dados de vendas.

- **Segmentação de Dados (Slicers):** Filtros dinâmicos por Ano, Mês, Categoria, Segmento e Região para uma exploração interativa.

- **Data Visualization:** Gráficos de barras, colunas e cards de KPI desenhados para facilitar a leitura rápida de resultados.

## 📂 Estrutura do Repositório
- ```README.md:``` Explicação do projeto, métricas e instruções.

- ```Dashboard_Vendas_EShop.xlsx:``` Arquivo Excel contendo a base de dados, cálculos e o dashboard finalizado.

## 🚀 Como utilizar
1. Baixe o arquivo ```.xlsx``` disponível neste repositório.

2. Utilize os filtros interativos no cabeçalho e na barra lateral para filtrar os dados por período ou categoria.

3. Os cards e gráficos serão atualizados automaticamente conforme sua seleção.
